# Git Basics

## git status

```bash
git status
```

Shows changed files.

## git add .

```bash
git add .
```

Adds all changed files.

## git commit

```bash
git commit -m "Add first Python exercise"
```

Creates a save point.

## git push

```bash
git push
```

Uploads changes to GitHub.

## git pull

```bash
git pull
```

Downloads latest changes from GitHub.

## git remote add origin

```bash
git remote add origin https://github.com/yourusername/backend-learning-day1.git
```

Connects your local project to GitHub.

## git branch -M main

```bash
git branch -M main
```

Renames your current branch to main.

## git push -u origin main

```bash
git push -u origin main
```

Uploads your project to GitHub for the first time.
